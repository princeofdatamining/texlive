TeX Live documentation

The principal documentation available on TeX Live:

  • [Our]TeX Live manual in English (also available as PDF) (RAW HTML OR RAW PDF).
  • Translations of the TeX Live manual:
    • [Our]Simplified Chinese(PDF) (RAW PDF)
    (Some versions may not be available if you did not install the full TeX Live.)

  • Some TeX Live web pages:
    • Documentation on install-tl and tlmgr.
      (Or run the programs with the sole option --help.)
    • Quick installation recipe (Unix-like systems).
    • Installation and other notes on Windows.

  • Finally, the texdoc program, installed with TeX Live, may be useful in finding and viewing documentation of packages and programs.
